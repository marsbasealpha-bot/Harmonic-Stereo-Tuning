# Harmonic Stereo Tuning

Professional-grade sonic frequency generator and stereo tuning suite.

## Features

- **Acoustic Precision Interface**: High-fidelity audio control with a sleek, dark-themed UI.
- **Stereo Gain Matrix**: Independent volume control for left and right channels.
- **Harmonic Generator**: Dual-oscillator tone generator for precise frequency tuning.
- **Oscilloscope Output**: Real-time visual feedback of audio waveforms.
- **PWA Support**: Installable as a standalone app on mobile and desktop devices.
- **Flexible Audio Sources**: Load local files or stream from remote URLs.
- **Interactive Knobs**: Custom-built rotary controls with dynamic sensitivity for fine-tuning.
- **Immersive Visuals**: Animated starfield background and responsive design.

## Tech Stack

- **Frontend**: React 18+, Vite, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Audio**: Web Audio API
- **PWA**: Service Workers & Web App Manifest

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   cd harmonic-stereo-tuning
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Usage

- **Playback Engine**: Use the transport controls (Play, Pause, Stop, Loop) to manage audio playback.
- **Upload/Load**: Use the "Upload File" or "Load URL" buttons to bring in your own audio content.
- **Tuning**: Adjust the knobs to change volume and frequency. The frequency knobs feature dynamic sensitivity—turn slowly for fine adjustments and quickly for rapid sweeps.
- **Installation**: Click the "Welcome" icon in the header to view the installation QR code and instructions for mobile devices.
